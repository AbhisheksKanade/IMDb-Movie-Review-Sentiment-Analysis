# 🎬 IMDb Movie Review Sentiment Analysis

A Machine Learning and Natural Language Processing (NLP) project that predicts whether a movie review is **Positive** or **Negative** using TF-IDF Vectorization and Logistic Regression.

## 🚀 Features

* Real-time sentiment prediction
* Interactive Streamlit web application
* NLP-based text preprocessing
* TF-IDF feature extraction
* Logistic Regression classifier
* Fast and accurate predictions
* User-friendly interface

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* NLTK
* Scikit-Learn
* Streamlit
* Matplotlib
* WordCloud

## 📊 Dataset

This project uses the IMDb 50,000 Movie Reviews Dataset, a widely used benchmark dataset for sentiment analysis tasks.

## ⚙️ Project Workflow

1. Data Preprocessing
2. Text Cleaning and Tokenization
3. TF-IDF Vectorization
4. Model Training using Logistic Regression
5. Sentiment Prediction
6. Result Visualization through Streamlit

## ▶️ Installation

```bash
git clone https://github.com/AbhisheksKanade/IMDb-Movie-Review-Sentiment-Analysis.git

cd IMDb-Movie-Review-Sentiment-Analysis

pip install -r requirements.txt
```

## ▶️ Run the Application

```bash
streamlit run app.py
```

* Feature Extraction: TF-IDF Vectorization
